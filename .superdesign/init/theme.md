# Design Tokens & Theme
The Flask application will adopt the **Organic Tech (Preset A)** theme tokens:
- Primary Color: Moss #2E4036
- Accent Color: Clay #CC5833
- Background Color: Cream #F2F0E9
- Text / Dark Color: Charcoal #1A1A1A
- Typography:
  - Headers: Plus Jakarta Sans / Outfit
  - Drama: Cormorant Garamond Italic
  - Monospace Data: IBM Plex Mono
