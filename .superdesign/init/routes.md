# Routes
Current FastAPI API endpoints:
- POST /recommend: recommendations for a user.
- GET /health: service health.
- GET /: root message.

Planned Flask UI routes:
- GET /: Renders the main dashboard interface.
- POST /recommend: API endpoint returning recommendations.
- GET /metrics: API endpoint returning model training and evaluation metrics.
- GET /config: API endpoint returning current pipeline hyperparameter config.
